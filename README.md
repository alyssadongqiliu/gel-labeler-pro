# Gel Labeler Pro

A professional web-based tool for labeling gel electrophoresis images.

## Features

- **Row & Band Labeling**: Easily mark rows and bands on your gel images.
- **Zoom Support**: Adjust image size from 10% to 300% for precise labeling.
- **Bilingual Interface**: Switch seamlessly between English and Chinese.
- **Keyboard Shortcuts**: Use `ESC` to cancel current operations.
- **Export Data**: Save your work as JSON or export labels as CSV.

## Usage

1. Open `index.html` in any modern web browser.
2. Click "Load Image" to upload your gel image.
3. Use the "Row" tool to define lanes.
4. Use the "Band" tool to mark specific bands.
5. Export your data when finished.
